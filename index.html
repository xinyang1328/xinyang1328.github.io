git命令

发表于 2019-06-10 | 分类于 其他

远程仓库

登录

连接

git remote add origin(远程仓库名) url

这里的url是仓库的地址（建议用ssh

克隆

git clone url

这个可以将仓库的代码拉下来，并且将本地仓库和远程仓库连接起来。

暂存

git stash：暂存更改

git stash list: 查看暂存列表

git stash apply: 恢复暂存内容

git stash drop: 删除暂存内容

git stash pop: 恢复暂存内容并删除

提交

git commit：将暂存区的内容提交到当前分支

推送

git push -u origin master: 第一次推送，-u可以将本地的master分支和远程的master分支关联起来

git push: 将当前分支推送到远程

git push 仓库名 分支名: 将该分支推送到远程仓库的分支

分支

git branch：查看所有分支，当前分支名前有 *

git branch Name：创建

git branch -d Name: 删除该分支(-D是强行删除)

git checkout Name：切换分支

git checkout -b Name: 创建并切换到分支

git merge Name: 将该分支和当前分支合并

git branch --set-upstream 分支名 仓库名/分支名：建立本地分支和远程分支的关联

git pull: 抓取分支

标签

git tag Name: 新标签

git tag -d Name: 删除本地标签

git push 仓库名 :refs/tags/Name: 删除远程标签

git show Name: 查看标签信息

git push 仓库名 Name: 将此标签推送到远程

git push 仓库名 -tags: 将未推送的标签推到远程

git checkout -- fileName: 撤销此文件在工作区的修改

其他

git log: 显示提交日志

git reflog: 显示命令历史

git reset --hard versonID: 历史版本切换

git status: 查看当前状态

git log --graph: 查看分支合并图

git remote [-v]: 查看远程库的信息

win10+VS2017+open CV 3.4.1配置

发表于 2019-01-06 | 分类于 其他

下载

1.VS下载：https://code.visualstudio.com/
2.open CV下载：https://opencv.org/

配置

1、环境变量
打开 电脑–>属性–>高级系统设置–>环境变量–>系统环境变量–>Path
将 XXX\opencv\build\x64\vc15\bin 路径添加到Path(XXX表示opencv所存放的位置)

2、VS中配置opencv
新建一个控制台项目，打开属性管理器

修改 VC++目录下的包含目录和库目录。
包含目录加上 XXX\opencv\build\include 、 XXX\opencv\build\include\opencv 和 XXX\opencv\build\include\opencv2

库目录加上 XXX\opencv\build\x\vc\lib

在 链接器 下的输入添加 opencv_world341d.lib

配置好后，点击 应用 ，配置完成。

测试

输入代码，能看到一张图片，表示配置成功。

配置失败的一个小问题

问题： 无法打开包括文件:“SDKDDKVer.h”: No such file or directory

解决方法：在 VC++ 目录下的 包含目录 和 库目录 里，分别添加 $(IncludePath) 和 $(LibraryPath)

图片叠加时的一个小问题

问题： 0xC0000005:写入位置 0x0000020AE9446212时发生访问冲突。

原因：图片叠加，将小照片放入背景时，小照片的较大，导致叠放失败。

解决：换用一张较小的照片放入背景或将背景图片的大小设置大一点。

1.量词：

①+：至少包含一个
②*:0个或多个
③？：0或1个
④ . :任意一个字符（几个 . 就代表几个任意字符）
⑤{x}：必须包含x个前导字符
⑥{x,y}：必须包含x到y个前导字符
⑦{x,}：必须包含至少x个前导字符
⑧$：从末尾开始匹配，末尾必须是模式
⑨^:从开头开始匹配
⑩|：有其中一个就可匹配
⑪():分组

结果

2.元字符

①[a-z]:匹配任何包含a-z的字符串
②[A-Z]:匹配任何包含A-Z的字符串
③[0-9]或\d:匹配任何包含0-9的字符串
④[abc]:匹配任何包含a、b、c的字符串
⑤[^abc]:匹配任何不包含a、b、c的字符串
⑥[a-zA-Z0-9_]或\w:匹配任何包含a-z、A-Z、0-9以及_的字符串
⑦\W:匹配任何没有下划线和字母数字的字符串
⑧\D:匹配任何不包含数字的字符串
⑨\s:匹配任何包含空白字符
⑩\S:匹配任何不包含空白字符
⑪\b:匹配是否达到边界
⑫\B:匹配是否没有达到边界
⑬\:匹配正则表达式的特殊字符(个人理解为转义)
